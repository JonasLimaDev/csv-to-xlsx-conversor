    Configurações para leitura do Arquivo de origem

    Estas configurações são relativas ao arquivo de origem (que se deseja converter), e serão aplicadas para a leitura correta das informações contidas nele.

    Codificação

    Tipo de codificação usada no Arquivo de origem. É o sistema utilizado para representar caracteres, o padrão é utf-8. Mas pode ser WINDOWS-1252 ou latin-1 por exemplo.

    Atenção: Somente uma codificação deve ser usada por vez

    Delimitador

    Delimitador das colunas do arquivo. O padrão ",". Existem outros delimitadores por exemplo ";" e "|".

    Configurações para Limpeza de Dados

    Estas configurações são relativas aos dados do arquivo de origem (que se deseja converter), e serão aplicadas nos dados salvos no arquivo de saída. Os dados do arquivo de origem permanecerão inalterados

    Configurações de Saída

    Salvar Novo Arquivo

    Decidir onde Salvar o Arquivo Processado. Por padrão será salvo com o nome do arquivo de origem. Ao habilitar essa opção será fornecida uma interface para nomear o arquivo de saída.

    Atenção: Arquivos com mesmo nome serão sobrescritos a cada novo processamento.
